Introducing Seed

Seed is a collection of small, useful HTML and CSS code snippets. Each snippet starts with default styles and design opinions while priorisiting giving you the chance to reuse, repurpose and customise everything you need.

Seed's helps you start simple projects as easily as possible. Just copy and paste whatever you need, changing things as you go.

Ready to get started? Just scroll on down. Time to build something rad!

Seed Design Goals:

  • Beginner friendly
  • Semantic HTML
  • Accessible HTML & CSS
  • Accessible Design
  • Responsive CSS & Design

Typography

Type Scale:

A x1.125 Major Second scale, calculated using https://type-scale.com/ and an 18px base.

Font Face:

System Font based on the website visitors operating system.

This scale was primarily designed to be used in Product Design, while a more pronounced type scale is recommended for Communication or Marketing Web Design (e.g Major Third).
