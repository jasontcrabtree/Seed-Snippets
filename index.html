Getting Started

Seed is a collection of small, useful HTML and CSS code snippets. Each snippet provides default styles, design and opinions to get you started while priorisiting giving you the chance to reuse, repurpose and customise everything you need.

Seed's main goal is providing defaults that help you get started, letting you build new projects, products and features as easily as possible.

Each snippet is simple, small and self-contained. The intention is that you can copy and paste whatever you choose, letting you use exactly what you need.

Ready to get started? Just scroll on down. Time to build something rad!

Seed Design Goals:

  • Beginner friendly
  • Semantic HTML
  • Accessible HTML & CSS
  • Accessible Design
  • Responsive CSS & Design
